# IOProdz

requirements: [Golang](https://go.dev/dl/)

### 🦺 install dev tools

```
sh .tooling/devtools.sh
```

### Setup config

```
cp .env.example .env
# Open .env in editor and setup the values
```

### 👷🏽 run for dev

```
fresh
```

### 🏗️ build and 🚜 run for prod

```
go build && ./ioprodz

```
